# Monetizao 🚀

Blog especializado en monetización con Inteligencia Artificial.

## 🎯 Misión

Ayudar a emprendedores digitales a generar ingresos sustanciales usando Inteligencia Artificial a través de estrategias probadas, herramientas lucrativas y casos de éxito.

## 🛠️ Tech Stack

- **Framework:** Next.js 14 con App Router
- **Lenguaje:** TypeScript
- **Styling:** Tailwind CSS
- **Temas:** next-themes (claro/oscuro)
- **Contenido:** MDX
- **SEO:** Next SEO
- **Icons:** Lucide React
- **UI Components:** Radix UI

## 🚀 Inicio Rápido

### Prerequisites

- Node.js 18+
- npm o yarn

### Instalación

1. Clonar el repositorio:
```bash
git clone <repository-url>
cd monetizao
```

2. Instalar dependencias:
```bash
npm install
```

3. Ejecutar el servidor de desarrollo:
```bash
npm run dev
```

4. Abrir [http://localhost:3000](http://localhost:3000) en tu navegador.

## 📝 Escribiendo Posts

Los posts se escriben en formato MDX en la carpeta `content/posts/`. Cada post debe incluir un frontmatter con los siguientes campos:

```mdx
---
title: "Título del Post"
description: "Descripción breve del contenido"
date: "2024-10-22"
author: "Nombre del Autor"
category: "IA para Negocios"
tags: ["ChatGPT", "Freelance", "Prompts"]
featured: true
readTime: 5
image: "/images/posts/ejemplo.jpg"
---

# Contenido del post...

Tu contenido en Markdown con soporte para componentes React.
```
